# Inventory System

Небольшая система инвентаря, оружия и сохранений на Unity 2022.3.

## Стек
- Unity 2022.3
- MVC-подобная модульная архитектура
- JSON save system через `persistentDataPath`

## Запуск
1. Открыть проект в Unity 2022.3 LTS.
2. Дождаться импорта пакетов и пересборки скриптов.
3. Запустить стартовую сцену из `Assets/Scenes`.
4. Нажать Play и проверить UI/сохранения.

## Архитектура
```text
Assets/
  Scripts/
    Core/            - общие сервисы и базовые типы
    Inventory/       - предметы, слоты, контейнеры
    Weapons/         - стрельба, патроны, логика оружия
    UI/              - drag&drop, окна, бинды интерфейса
    Save/            - сериализация и загрузка данных
    Economy/         - монеты и разблокировка
  Prefabs/           - UI и игровые префабы
  ScriptableObjects/ - данные предметов и оружия
  Scenes/            - игровые сцены
```

## Что умеет
- Инвентарь с набором слотов
- Перетаскивание предметов мышкой
- Стак патронов по типу
- Логика выстрела и расхода
- Сохранение и загрузка
- Разблокировка за монеты

## Время выполнения
12 часов
